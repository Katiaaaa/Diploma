Для запуску веб-додатку на Flask, необхідно відкрити термінал або командний рядок і виконати наступні кроки:

Перейдіть у директорію з вашим проектом.
Створіть віртуальне середовище для Python. Виконавши команду python3 -m venv venv. 
Ця команда створить папку з назвою "venv", яка буде містити всі необхідні залежності для вашого веб-додатку.

Активуйте віртуальне середовище за допомогою команди venv\Scripts\activate на Windows.
Встановіть Flask за допомогою команди pip install flask.
python.exe -m pip install --upgrade pip
Запустіть веб-додаток за допомогою команди flask run.
Деактивуйте за допомогою deactivate

WebSite.py = app.py
FLASK_APP=app.py

flask --app api.py run
